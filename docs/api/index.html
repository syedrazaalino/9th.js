Overview

9th.js is a modern 3D JavaScript library for creating interactive graphics and visualizations in web browsers using WebGL. It provides a comprehensive set of classes for 3D rendering, animation, geometry, materials, lighting, and more.

API Reference

Browse the complete 9th.js API documentation organized by category.

Core API

Essential classes for 3D scene management, rendering, and object handling.

  • Engine - Main application controller
  • Scene - 3D scene container
  • WebGLRenderer - Rendering system
  • Object3D - Base 3D object class
  • BufferGeometry - Geometry storage

Cameras

Camera systems for different projection types and viewing perspectives.

  • Camera - Base camera class
  • PerspectiveCamera - Perspective projection
  • OrthographicCamera - Orthographic projection
  • CubemapCamera - 360¬∞ environment capture

Geometry

3D geometric shapes and geometry utilities.

  • BoxGeometry - Box-shaped geometry
  • SphereGeometry - Sphere-shaped geometry
  • PlaneGeometry - Flat plane geometry
  • CylinderGeometry - Cylinder geometry
  • ConeGeometry - Cone geometry

Materials

Material systems for controlling object appearance and rendering.

  • Material - Base material class
  • BasicMaterial - Unlit material
  • PhongMaterial - Phong shading
  • StandardMaterial - PBR standard material
  • PhysicalMaterial - Advanced PBR

Lighting

Lighting systems for illuminating 3D scenes.

  • Light - Base light class
  • AmbientLight - Ambient lighting
  • DirectionalLight - Sun-like light
  • PointLight - Point light source
  • SpotLight - Spotlight with angle

Animation

Animation system for creating dynamic, interactive 3D content.

  • AnimationClip - Animation data
  • AnimationMixer - Animation controller
  • KeyframeTrack - Keyframe animation
  • AnimationAction - Animation instance

Loaders

Asset loading utilities for textures, 3D models, and data files.

  • Loader - Base loader class
  • TextureLoader - Texture loading
  • GLTFLoader - glTF format support
  • OBJLoader - OBJ format support

Math & Utils

Mathematical utilities and helper classes for 3D operations.

  • Vector3 - 3D vector operations
  • Matrix4 - 4x4 matrix operations
  • Quaternion - Quaternion operations
  • Color - Color utilities
  • MathUtils - Math helper functions

Controls

User interaction controls for cameras and objects.

  • OrbitControls - Orbital camera control
  • FirstPersonControls - FPS camera control
  • DragControls - Object dragging
  • TransformControls - Transform gizmos

Rendering

Advanced rendering techniques and post-processing effects.

  • ShaderMaterial - Custom shaders
  • EffectComposer - Post-processing
  • RenderPass - Rendering passes
  • ShaderPass - Shader passes

Particles

Particle systems for creating effects like fire, smoke, and sparkles.

  • ParticleSystem - Particle emitter
  • Particle - Individual particle
  • ParticleMaterial - Particle material
  • ParticleEmitter - Particle controller

Physics

Physics simulation for realistic object interactions.

  • PhysicsSystem - Physics engine
  • RigidBody - Physics object
  • CollisionShape - Collision detection
  • Joint - Object constraints
